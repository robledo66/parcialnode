Bienvenido a la API de Películas

Descripción

Esta es una API RESTful para gestionar una colección de películas. A través de ella pueder ver todas las películas, buscar por ID, agregar, actualizar y eliminar peliculas.

Endpoints disponibles

  • peliculas - Obtener todas las peliculas
  • peliculas/:id - Obtener una película por id
  • directores - Obtener todos los directores
  • directores/:id - Obtener un director por id
